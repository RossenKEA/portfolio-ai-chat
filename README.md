# Portfolio AI Chat

A simple AI chat application built with Next.js, TypeScript, Tailwind CSS, and Google Gemini.

## Features

- Chat interface built with React and Next.js App Router
- Server-side API route for AI requests
- Google Gemini integration
- Transparent demo fallback mode
- Automatic demo mode after 3 user messages to protect API quota
- Local chat history using localStorage
- Clear chat button
- Copy response button
- Markdown-rendered AI responses

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Vercel AI SDK
- Google Gemini API
- React Markdown
